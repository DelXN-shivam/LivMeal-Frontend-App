LivMeal Android App Icon Generator

Instructions:

  1. Choose your preferred blue gradient by clicking on the color options above
  2. Click "Change App Text" to customize the text on your icon
  3. Click "Download App Icon" to save the 1024x1024 PNG file
  4. Save the downloaded file as app_icon.png in your assets/images/ folder
  5. Run the Flutter commands provided below to generate all icon sizes

After downloading the icon:

Run these commands in your Flutter project directory:

flutter pub get
flutter pub run flutter_launcher_icons
